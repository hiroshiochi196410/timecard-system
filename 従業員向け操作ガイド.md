# 📱 タイムカード打刻アプリ 簡単操作ガイド

## 初回設定（デバイス登録）

### ステップ1: アプリを開く
スマホでアプリのURLにアクセスします。

### ステップ2: 社員番号を入力
```
┌─────────────────────┐
│  社員番号4桁を入力  │
│                     │
│    [ 1 2 3 4 ]     │  ← 半角数字で入力
│                     │
│   [登録する]        │
└─────────────────────┘
```

**注意点**:
- ✅ 半角数字で入力: `1234`
- ❌ 全角数字はNG: `１２３４`
- ✅ 4桁ちょうど: `0101`（0も含めて4桁）

### ステップ3: 登録完了
```
✅ 〇〇さんのデバイスを登録しました
```
このメッセージが出たら成功です！

## よくある質問

### Q1: スマホを変えたらどうする？
A: 新しいスマホで同じ手順で登録し直してください。

### Q2: 間違って打刻した場合は？
A: 管理者に連絡して修正を依頼してください。

### Q3: 家のPCでも使える？
A: はい、PCでも使えます。各デバイスごとに登録が必要です。

### Q4: プライベートモードで使える？
A: いいえ、通常モードで使用してください。

## 🆘 困ったときは

### 「データ保存機能が利用できません」と出る
→ プライベートモード（シークレットモード）を終了して、通常モードで開いてください

### 「社員番号が見つかりません」と出る
→ 社員番号が正しいか確認してください（半角数字4桁）

### それでも解決しない場合
→ 管理者に以下を伝えてください：
- 使用しているスマホの機種
- 表示されたエラーメッセージ
- 社員番号

---

**重要**: このアプリはインターネット接続が必要です。打刻前にWi-Fiまたはモバイルデータ通信がオンになっているか確認してください。
